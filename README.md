## Hakkinda
Bu proje, Web Tasarimi ve Programlama dersi LAB-1 ve LAB-2 kapsaminda Vite + React + TypeScript kullanilarak olusturulmustur.
Ayni zamanda Semantik HTML5 ve Erisilebilirlik (a11y) standartlarina uygun sekilde yapilandirilmistir.
